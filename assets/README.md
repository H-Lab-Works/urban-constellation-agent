# Demo Video

`demo-video.mp4` is too large for normal GitHub file upload (25MB web limit, 100MB git limit).

Upload it as a **Release asset** instead:

1. Open the repository on GitHub.
2. Go to **Releases** → **Draft a new release**.
3. Choose tag `demo-video-v1` (or create a new tag).
4. Attach `demo-video.mp4` under **Attach binaries**.
5. Publish the release.

After publishing, the main README video link will work.
